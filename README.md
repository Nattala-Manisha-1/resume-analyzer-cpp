# Smart Resume Analyzer (ATS Simulation) - C++

## 📌 Overview
This project is a C++ based Smart Resume Analyzer that evaluates resumes against role-specific skill sets and provides an ATS-like score along with improvement suggestions.

## 🚀 Features
- Role-based analysis (Software Developer / Data Analyst)
- Keyword extraction using string processing
- Frequency-based skill matching
- Weighted scoring system (0–100)
- Missing skill suggestions

## 🛠️ Tech Stack
- C++
- File Handling
- String Processing
- STL (Vectors, Strings)

## 📂 Project Structure
resume-analyzer/
│── resume_analyzer.cpp
│── resume.txt
│── README.md

## ▶️ How to Run
1. Compile:
   g++ resume_analyzer.cpp -o analyzer

2. Run:
   .\analyzer.exe

## 💡 Example Output
- Score: 70/100
- Skills Found: C++, Python
- Missing Skills: SQL, Machine Learning
